# Internet of Toys

##Infrastructure

###Devices
- Arduino Uno (AVR ATmega16U2)
- Raspberry Pi (ARM Cortex-A53)
- Parts

###IoT platform
- Apache ZooKeeper
- Apache Kafka
- Apache Cassandra
- Apache Spark

###Networking

- Inter-device: GPIO/SPI/I2C/ZigBee/BLE/USB
- Device to IoT platform: Ethernet and WiFi
